## Components

Nextbase ultimate comes with Fumadocs built in. Which means you can use all the components and features that Fumadocs provides including the following:

- Banner
- Image with zoom
- Folder view
- Code blocks
- Text
- Callouts
- Accordion
- Tabs
- ... and more!
